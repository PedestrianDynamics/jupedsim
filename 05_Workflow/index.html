Workflow

  1. Clone project
    git clone https://cst.version.fz-juelich.de/jupedsim/jpscore.git
  1. Create branch with your feature
    git checkout -b feature_name
  1. Write code. Commit changes
    git commit -am "My feature is ready"
  1. Push your branch to GitLab. But before doing so, test your code.
    git push origin feature_name

  1. Review your code on Commits page

  2. Create a merge request

Your team leader will review code & merge it to the main branch.
