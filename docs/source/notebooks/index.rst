=========
Notebooks
=========

The notebooks are organised into three groups by intent:

* **Fundamentals** -- API-first tutorials. Read these to learn how the
  parts of JuPedSim fit together.
* **Scenarios** -- "solve a problem" studies that use one notebook to
  investigate a phenomenon (lane formation, single-file flow, model
  comparison, ...).
* **Standards** -- one-notebook-per-test implementations of pedestrian
  dynamics test standards (RiMEA, ...).
