*************
Cite JuPedSim
*************


If you use *JuPedSim* in your work, please cite it using the following information from Zenodo.

.. literalinclude:: jupedsim_bibtex.bib
    :language: bibtex

Information to all versions of *JuPedSim* can be found on `Zenodo <https://zenodo.org/doi/10.5281/zenodo.1293771>`_.

.. image:: https://zenodo.org/badge/DOI/10.5281/zenodo.1293771.svg
    :target: https://doi.org/10.5281/zenodo.1293771

To find your installed version of *JuPedSim*, you can run:

.. code-block:: bash

    import jupedsim
    print(jupedsim.__version__)
