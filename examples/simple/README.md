# Simple starter scripts

Plain-Python, copy-paste-runnable end-to-end examples for each core API
surface. No Jupyter, no plotting helpers, no scenario layer. Each script
mirrors a notebook in `notebooks/fundamentals/`.

| Script | Mirrors |
|---|---|
| `01_basic_simulation.py` | `fundamentals/00_getting_started` |
| `02_journey.py` | `fundamentals/07_journeys_transitions` |
| `03_routing.py` | `fundamentals/08_routing` |
| `04_queues_waiting.py` | `fundamentals/05_queues`, `06_waiting` |
| `05_direct_steering.py` | `fundamentals/10_direct_steering` |

Run any of them with the project venv active:

    python examples/simple/01_basic_simulation.py
